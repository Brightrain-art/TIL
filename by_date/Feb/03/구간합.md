## 구간합 구하기 문제

- 무난한 문제였습니다. 개인적으로 평소 문제 풀이를 할 때 슬라이싱 활용을 하지 못하는 게 아쉬웠는데 이제 자연스럽게 문제해결에 활용할 수 있게 되었다는 점이 기쁩니다.

- 이전에 min max 값 설정을 할 때 음수값을 상정하지 못해 데인적이 있기 때문에 이번에는 문제에서 정의가 되어있지만 그래도 신경써서 풀어봤습니다.

```python
TC = int(input())

for test_case in range(1, TC + 1):
    pass
    N, M = map(int, input().split())
    nums = list(map(int, input().split()))

    # nums 에서 M 만큼 부분 합을 구한다.
    # 슬라이싱?
    # idx + M 이 N 을 넘어가면 break

    # min, max 설정
    # 문제에 정수 범위는 1 <= a <= 10000으로 정의되어 있지만 그렇지 않을 때는 음수값도 상정해야함
    sum_max_M = -(10**99)
    sum_min_M = 10**99

    for idx in range(N):
        if idx + M > N:
            break

        # min max 구하기
        if sum(nums[idx: idx + M]) > sum_max_M:
            sum_max_M = sum(nums[idx: idx + M])

        if sum(nums[idx: idx + M]) < sum_min_M:
            sum_min_M = sum(nums[idx: idx + M])

    print(f'#{test_case} {sum_max_M - sum_min_M}')
```
