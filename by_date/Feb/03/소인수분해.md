## 소인수분해

- 문제에서 소인수분해도 해주고 다 줘서 계산만 하면 되는 문제였습니다.



```python
import sys
sys.stdin = open('nums_input.txt')

# 소인수분해 문제
# 소수를 다 줘서 그냥 값만 구하면 될 것 같음

TC = int(input())  # 테스트 케이스

for test_case in range(1, TC + 1):  # 순회
    N = int(input())  # N 입력
    num_lst = [2, 3, 5, 7, 11]  # 주어진 소수

    result_lst = []  # 결과 담을 그릇
    for num in num_lst:  # 주어진 소수 순회하며 계산
        cnt = 0  # 나눠질 때마다 카운트 추가 및 다음 소수로 가면 초기화
        while N % num == 0:  # 나머지가 0이면 계속하고 나머지가 생기면 다음 수로 넘어가는 while 문
            N = N / num  # 소수로 나누기
            cnt += 1  # 나눈 횟수 카운트
        result_lst.append(cnt)  # while 문 종료되면 카운트 결과 리스트에 추가

    a, b, c, d, e = result_lst  # 각 회수를 변수에 할당

    print(f'#{test_case}', a, b, c, d, e)  # 출력
```
