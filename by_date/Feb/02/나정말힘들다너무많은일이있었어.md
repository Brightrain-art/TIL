# SWEA - 4828
## MIN, MAX 문제
```python
TC = int(input()) # 테스트 케이스

for test_case in range(1, TC + 1): # 테스트 케이스 만큼 반복

    N = int(input()) # 정수 개수

    num_list = list(map(int, input().split())) # 정수 리스트 인풋

    for right in range(len(num_list)-1, 0, -1): # 방울분류 로직
        for idx in range(right):
            if num_list[idx] > num_list[idx + 1]:
                num_list[idx], num_list[idx + 1] = num_list[idx + 1], num_list[idx]

    answer = num_list[-1] - num_list[0] # 오름차순이니까 인덱스 -1이 최대 0이 최소

    print(f'#{test_case} {answer}') # 출력
```
- 오늘 배운 버블정렬 로직을 이용해서 풀이해봤습니다. 무난하게 풀이했고 학습한 내용을 이용해 해결했기에 별다른 후기가 없습니다.

- 굳이 하나 남기자면 버블정렬 이론 페이지에 있던 플래그 변수를 활용한 최적화를 한 번 해보고 싶습니다. 하게 된다면 추후 추가하겠습니다.



# 강사님의 Solution
## Solution no.1
```python
import sys
sys.stdin = open('input.txt')

T = int(input())

for tc in range(1, T+1):
    N = int(input())
    arr = list(map(int, input().split()))

    max_v = arr[0]
    min_v = arr[0]

    for i in range(len(arr)):
        if i == 0:
            continue
        if arr[i] > max_v:
            max_v = arr[i]
        if arr[i] < min_v:
            min_v = arr[i]

    result = max_v - min_v
    print(f'#{tc} {result}')
```
- 리스트를 순회하며 최대값 최소값을 동시에 구하는 매우 영리한 로직이라는 생각이 듭니다. 알아두면 이모저모 사용할 곳이 많아보입니다.

import sys
